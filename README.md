# The Shobha Mother Foundation 

This is the official website of **The Shobha Mother Foundation**, an NGO started by Shobha from rural Kota.

## About
Our mission is to educate underprivileged children, provide skill development, and promote environmental awareness through tree planting and community programs.

## Features
- Information about our NGO
- Our mission and work
- Gallery for photos
- Contact details
- Donation and volunteering guidance

## How to Use
1. Open `index.html` in any browser.
2. Replace placeholder images in the `img` tags with your own images.
3. Update contact details if needed.
